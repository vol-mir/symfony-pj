# Symfony PJ

## About Project
News Parser. The application is developed using Symfony.
http://127.0.0.1:8080/

## Quick Start
```shell
$ cd symfony-pj
$ cp app/.env.example app/.env
$ docker-compose up --build
$ docker exec -it php74-container bash
$ composer install
$ sh makedb.sh
```
